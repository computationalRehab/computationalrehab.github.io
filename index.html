Welcome to ComputationalRehab

ComputionalRehab.github.io is the web interface to the GitHub repositories hosting code related to computational aspects of neurorehabilitation. This webpage provides links to all the relevant information and code hosted in the respositories, along with any supplementary information provided by the contributors.

ComputationalRehab has been setup as part of COST Action TD1006 - European Network on Robotics for NeuroRehabilitation.

Current repositories

The following are the currently maintained repositories @ComputationalRehab.

  • Quantitative measures of motor performance
  • Models of motor recovery
